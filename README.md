# GP Valise API

API REST sécurisée en Laravel 12 pour la mise en relation entre voyageurs et expéditeurs de valises.

## Stack actuelle

-   Laravel 12
-   Sanctum (authentification API)
-   PestPHP (tests)
-   MySQL (Docker à venir)
-   CI/CD GitHub Actions (à venir)

## Modèles existants

-   Trip
-   Luggage
-   Booking
-   Payment
-   Report
